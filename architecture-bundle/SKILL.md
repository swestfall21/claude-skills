# Architecture bundle

Use this bundle when shaping systems, proposing major design changes, or evaluating architectural trade-offs.

## Activated skills

When this bundle is invoked, read each of the following skills' SKILL.md files and apply them for the rest of the session:

- architect-mode
- decision-making-guardrails
- review-quality-gate
- migration-refactor-safety

## Operating posture

- frame the problem around outcomes and constraints
- make trade-offs explicit
- evaluate the long-term impact of the design
- protect the change with a migration and rollback strategy
